# 📘 Day 05 – Linux Text Processing Tools (find, awk, sed)

## 🚀 Introduction

Today explored powerful Linux tools widely used in real-world DevOps environments:

- find
- awk
- sed

These tools are essential for automation, log analysis, and system management.

---

## 🔎 1. find Command

The find command is used to search files and directories efficiently.

It is very useful in:
- Locating log files
- Finding configuration files
- Searching large directory structures
- Automating file cleanup tasks

It can search based on:
- File name
- File type
- Size
- Time
- Permissions

---

## 📊 2. awk Command

awk is a powerful text processing tool.

It works mainly with:
- Columns
- Patterns
- Structured text data

Common use cases:
- Extract specific columns from files
- Analyze log files
- Generate reports
- Process CSV or structured data

awk is widely used in DevOps for log analysis and data extraction.

---

## ✏️ 3. sed Command

sed is a stream editor used to modify text.

It is commonly used for:
- Replacing text
- Updating configuration values
- Removing lines
- Editing files automatically

Very helpful in automation scripts and deployment tasks.

---

## 🧠 Real-World DevOps Usage

These tools help in:

- Log analysis
- Automating repetitive tasks
- Managing large systems
- Updating configuration files automatically
- Processing large data efficiently
