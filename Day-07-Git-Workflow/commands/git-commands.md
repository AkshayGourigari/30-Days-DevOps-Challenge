# ⚙️ Git Commands Cheat Sheet

---

## 📌 Repository Setup

git clone <repo-url>

---

## 📌 Checking Changes

git status
git diff

---

## 📌 Adding & Committing

git add .
git commit -m "message"

---

## 📌 Pushing & Pulling

git push origin main
git pull origin main

---

## 📌 Branching

git checkout -b feature-branch
git branch
git checkout main

---

## 📌 Merging

git merge feature-branch

---

## 📌 Logs

git log

---

## 📌 Quick Workflow

git add .
git commit -m "update"
git push origin main
