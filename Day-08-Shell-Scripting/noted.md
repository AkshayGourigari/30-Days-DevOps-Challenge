## 📌 Topic
Shell Scripting for Automation in Linux (AWS EC2)

---

## 📌 Objective
To understand and implement shell scripting for automating repetitive tasks in a Linux environment, which is a core skill in DevOps.

---

## 📌 Introduction

Shell scripting is the process of writing a sequence of commands in a file (script) that can be executed to perform tasks automatically.

Instead of running commands manually one by one, we can write them in a script and execute them whenever needed.

---

## 📌 Why Shell Scripting is Important in DevOps

- Automates repetitive manual tasks  
- Saves time and improves efficiency  
- Reduces human errors  
- Helps in server management and monitoring  
- Used in CI/CD pipelines and deployment processes
