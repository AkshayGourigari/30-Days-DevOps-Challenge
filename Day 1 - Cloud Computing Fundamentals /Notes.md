# 📘 Day 01 – Cloud Computing Notes

## ☁️ What is Cloud Computing?

Cloud computing is the delivery of computing services such as servers, storage, databases, networking, and software over the internet instead of maintaining physical infrastructure.

It helps organizations:
- Reduce infrastructure cost
- Scale applications easily
- Improve performance and availability
- Pay only for what they use

---

## 🌍 Types of Cloud Deployment Models

### 1️⃣ Public Cloud
- Infrastructure owned by cloud provider
- Services available over internet
- Example: AWS, Azure, GCP
- Highly scalable and cost-effective

### 2️⃣ Private Cloud
- Dedicated infrastructure for single organization
- Higher control and security
- Used by enterprises with compliance needs

### 3️⃣ Hybrid Cloud
- Combination of public + private cloud
- Flexible and optimized environment

---

## 🏗️ Cloud Service Models

### 🔹 IaaS (Infrastructure as a Service)
- Provides virtual machines, storage, networking
- User manages OS, middleware, and applications
- Example: AWS EC2

### 🔹 PaaS (Platform as a Service)
- Provides platform to develop, run applications
- No need to manage servers
- Example: Heroku

### 🔹 SaaS (Software as a Service)
- Ready-to-use software over internet
- No infrastructure management needed
- Example: Gmail

---

## 🧠 Key Differences

| Feature | IaaS | PaaS | SaaS |
|---------|------|------|------|
| Infrastructure | Managed by provider | Managed by provider | Managed by provider |
| OS Management | User | Provider | Provider |
| Application | User | User | Provider |
| Example | EC2 | Heroku | Gmail |

---

## 🎯 Conclusion

Cloud computing enables scalable, flexible, and cost-efficient infrastructure for modern applications.

This is the foundation for DevOps and cloud-native architecture.
