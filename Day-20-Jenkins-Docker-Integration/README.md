# 🚀 Day 20 – Jenkins + Docker CI/CD Pipeline

## 📌 Project Overview
This project demonstrates CI/CD pipeline using:
- Jenkins
- Maven
- Docker

---

## ⚙️ How it Works

1. Jenkins pulls code from GitHub  
2. Maven builds project  
3. Docker builds image  
4. Container runs automatically

Automated build and deployment pipeline.
