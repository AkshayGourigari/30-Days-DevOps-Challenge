# 💻 Day 02 – Linux Commands (Practice & Explanation)

## 📍 1. Check Current Directory

pwd

Shows the current working directory.

---

## 📂 2. List Files and Directories

ls
ls -l
ls -a

- ls → List files
- ls -l → Detailed view
- ls -a → Show hidden files

---

## 📁 3. Change Directory

cd foldername
cd ..
cd /

- cd → Enter directory
- cd .. → Go back one level
- cd / → Go to root directory

---

## 📦 4. Create Directory

mkdir devops
mkdir -p project/demo

- mkdir → Create folder
- -p → Create parent directories if not exist

---

## 📄 5. Create File

touch file.txt

Creates an empty file.

---

## ❌ 6. Remove Files and Directories

rm file.txt
rm -r foldername

- rm → Delete file
- rm -r → Delete directory

---

## 📋 7. View File Content

cat file.txt

Displays file content.

---

## 🧪 Practice Example

pwd
ls
mkdir linux-practice
cd linux-practice
touch test.txt
ls -l
cat test.txt
cd ..
rm -r linux-practice

---
