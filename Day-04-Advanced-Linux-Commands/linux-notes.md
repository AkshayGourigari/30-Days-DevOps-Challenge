# 📘 Day 04 – Advanced Linux Commands

## 🚀 Introduction

Today focused on advanced Linux commands that are very useful in real-world DevOps tasks such as log analysis, debugging, automation, and data filtering.

---

## 🔍 1. grep Command

The grep command is used to search text inside files.

It is widely used in:
- Log analysis
- Error searching
- Debugging
- Monitoring scripts

### Important Options:

- -i → Ignore case
- -n → Show line numbers
- -v → Invert match
- -r → Search recursively in directories

---

## 🔗 2. Pipe ( | )

Pipe allows combining multiple commands.

It sends the output of one command as input to another command.

Example use cases:
- Filtering logs
- Counting matches
- Processing command outputs

Pipes make Linux extremely powerful for automation.

---

## 📄 3. cat Command

Used to:
- Display file content
- Combine files
- Create files (basic use)

---

## 📝 4. echo Command

Used to:
- Print text to terminal
- Create files
- Add content to files
- Append data

Very useful in scripting and automation.
